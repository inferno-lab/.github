**_Inferno Lab_ is a suite of 3 related security testing products.**

### Product Suite

**Apparatus:** Network and security simulation and diagnostic platform. _(aka Apts)_

**Crucible:** Offensive security simulator with 125+ built-in attack
scenarios defensive security testing with enterprise reporting and analytics.
_(aka Cruc)_

**Chimera:** Exploit lab and multi-vertical vulnerable attack
surface spanning 25+ industries. _(aka Chim)_

_Other Tools & Utilities - A command-line interface is available for Apparatus_
